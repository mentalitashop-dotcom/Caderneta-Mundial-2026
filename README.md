Caderneta Mundial 2026
======================

Como abrir:
1. Extrai o zip todo para uma pasta.
2. Abre o ficheiro abrir_caderneta.bat.
3. A app abre no browser em http://localhost:1312.
4. Quando fechares a pagina da app, a janela do servidor fecha sozinha.

Importante:
- Nao precisas de instalar Node.js. Esta pasta ja leva um Node portatil em runtime\node.exe.
- Nao apagues a pasta runtime.
- Na primeira abertura, a app cria automaticamente o ficheiro cromos.txt.
- O ficheiro cromos.txt guarda os cromos obtidos e repetidos dessa pessoa.

Ficheiros:
- abrir_caderneta.bat: abre a app.
- runtime\node.exe: Node portatil usado pela app.
- caderneta_mundial_2026.html: interface da app.
- server.js: servidor local que guarda o cromos.txt.
- cromos_base.txt: caderneta vazia usada para criar o cromos.txt.
